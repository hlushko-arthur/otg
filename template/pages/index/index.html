Welcome to WAW Template
Readme:
To create new page you can type in terminal: waw page NAME
JSON in template.json is accesible via html if {"port": 8000} then {{{port}}} will show 8000
JSON in page.json is accesible via html if {"title": "Cool Title"} then {{{title}}} will show Cool Title
More details of how to use template you can find in Wjst Documentation
Styles: You should import each file to index.scss located in css folder and write SCSS files in css/scss.
Important! You have global styles in css/scss. You can write separated styles in pages/title/index.scss
